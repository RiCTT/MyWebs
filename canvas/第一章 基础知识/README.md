### 7、关于求x点和y点的坐标
  > 最后Math.cos(x) x是一个弧度的值
  - x = radius * cos(angle)
  - y = radius * sin(angle)
### 8、关于canvas的元素大小和绘制表面大小
  - 绘制表面大小会自动根据canvas元素大小进行适配
    - 比如元素大小600 * 300，绘制表面大小300 * 150
    - 这个时候绘制的元素都会被放大，宽放大2倍，高也是2倍
  - 默认在元素上设置的是canvas的元素和绘制表面大小
  - css中设置的是元素的大小
